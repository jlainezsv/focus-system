# Focus System (MVP)

Sistema personal minimalista para constancia diaria, no-negociables y revisión sin culpa.

## Stack

- Vite + React + TypeScript
- Estado global ligero + `localStorage`
- PWA básica (manifest + service worker)
- CSS simple, texto > iconos

## Requisitos

- Node.js 18+
- npm o pnpm o yarn

## Instalación

```bash
npm install
npm run dev
